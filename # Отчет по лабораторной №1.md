# Задание



Выделить на GPU массив arr из 10^9 элементов типа float и инициализировать его с помощью ядра следующим образом: arr\[i] = sin((i%360)\*Pi/180). Скопировать массив в память центрального процессора и посчитать ошибку err = sum\_i(abs(sin((i%360)\*Pi/180) - arr\[i]))/10^9. Провести исследование зависимости результата от использования функций: sin, sinf, \_\_sin. Объяснить результат. Проверить результат при использовании массива типа double.



### Характеристики устройства

&nbsp; Device Name: Quadro RTX 4000

&nbsp; Total Global Memory (bytes): 8167620608

&nbsp; Shared Memory per Block (bytes): 49152

&nbsp; Compute Capability: 7.5

&nbsp; Max Threads per Block: 1024

&nbsp; Multiprocessor Count: 36

&nbsp; Clock Rate (kHz): 1545000

&nbsp; Warp Size: 32

&nbsp; Registers Per Block: 65536

&nbsp; Memory Pitch (bytes): 2147483647

&nbsp; ECC Enabled: No



### Результат выполнения программы:

Error using \_\_sinf (float): 8.01664e-08 | Time: 19.2553 ms

Error using sinf (float): 8.3571e-09 | Time: 18.7772 ms

Error using sin (double): 8.77963e-18 | Time: 43.977 ms



### Выводы:

**1. Точность вычислений:** Для типа float ошибки составляют порядка 10⁻⁸, в то время как для типа double ошибка значительно меньше — порядка 10⁻¹⁸. Это подтверждает, что использование типа double обеспечивает более высокую точность вычислений по сравнению с float.

**2.Время выполнения:** Для sin (double) время выполнения значительно больше по сравнению с другими функциями — около 43 мс. Это может свидетельствовать о том, что функции для работы с типом double требуют больше ресурсов и времени, что является нормальным явлением, учитывая их повышенную точность.
